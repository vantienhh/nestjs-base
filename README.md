## Environment
- node (v14.5)
- mongodb
- redis (optional)
- fireStore (optional)

## Install
```bash
$ yarn install
```

## Running the app

```bash
# lint
$ yarn lint

# development
$ yarn start

# watch mode
$ yarn start:dev

# watch debug
$ yarn start:debug

# build
$ yarn build

# production mode
$ yarn start:prod
```

## Environment
- node (v14.5)
- mongodb
- redis (optional)
- fireStore (optional)

## Install
```bash
$ npm install
```

## Running the app

```bash
# lint
$ npm run lint

# development
$ npm run start

# watch mode
$ npm run start:dev

# watch debug
$ npm run start:debug

# build
$ npm run build

# production mode
$ npm run start:prod
```
